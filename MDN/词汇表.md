# Glossary

## Bun

基于 Apple 的 JavaScriptCore 构建

## 引擎

- JavaScript 引擎(解析并执行 JavaScript 程序)
  - V8
  - SpiderMonkey
  - JavaScriptCore
- 渲染引擎
  - Blink
  - WebKit
  - Gecko

## Fetch metadata request header

- Sec-Fetch-Site
- Sec-Fetch-Mode
- Sec-Fetch-User
- Sec-Fetch-Dest

上面的标头不能通过 JavaScript 修改

## Long Task

需要超过 50ms 完成的任务

## 无损压缩

gzip，WebP, PNG, Zstandard

## Random

- `crypto.getRandomValues(typedArray)` 提供密码学安全的数字
- `crypto.randomUUID()` 生成 UUID v4

```js
// 生成一个随机数
const arr = new Uint32Array(1)
crypto.getRandomValues(arr)
console.log(arr[0])
```

## Document

- createElement() 创建HTMLUnkonwnElement 元素

## DOM

- Element
  - classList
  - clientTop/Left/Width/Height 不包括外边距和内边距  
  - scrollTop/Left/Width/Height
- Event
- Node

## HTML DOM API

- HTMLElement
  - dataset
  - offsetTop/Left 当前元素相对其 offsetParent 元素的顶部内边距的偏移量
  - offsetWidth/Height 元素 CSS 宽度的像素测量值，包括任何边框、内边距和垂直滚动条（如果已渲染）
- HTMLCanvasElement
  - getContext() 获取渲染上下文 [CanvasRenderingContext2D](https://developer.mozilla.org/zh-CN/docs/Web/API/CanvasRenderingContext2D)
  - toDataURL() 将画布内容转换为数据 URL
  - toBlob() 将画布内容转换为 Blob 对象

## UI Event

- MouseEvent
  - screenX/Y 鼠标在屏幕上的坐标
  - clientX/Y(x/y) 鼠标在元素上的坐标
  - pageX/Y 鼠标在页面上的坐标(包含滚动距离)
  - offsetX/Y 事件对象与目标节点的内部填充的偏移量
