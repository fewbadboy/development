# Glossary

## Bun

基于 Apple 的 JavaScriptCore 构建

## 引擎

- JavaScript 引擎(解析并执行 JavaScript 程序)
  - V8
  - SpiderMonkey
  - JavaScriptCore
- 渲染引擎
  - Blink
  - WebKit
  - Gecko

## Fetch metadata request header

- Sec-Fetch-Site
- Sec-Fetch-Mode
- Sec-Fetch-User
- Sec-Fetch-Dest

上面的标头不能通过 JavaScript 修改

## Long Task

需要超过 50ms 完成的任务

## 无损压缩

gzip，WebP, PNG, Zstandard

## Random

- `crypto.getRandomValues(typedArray)` 提供密码学安全的数字
- `crypto.randomUUID()` 生成 UUID v4

```js
// 生成一个随机数
const arr = new Uint32Array(1)
crypto.getRandomValues(arr)
console.log(arr[0])
```
