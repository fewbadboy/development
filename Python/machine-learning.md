# Machine Learning

## 标准差

$$
\sigma = \sqrt{\frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2}
$$

## Supervised Learning (监督学习)

线性回归、逻辑回归
支持向量机（SVM）
决策树与随机森林
K 近邻（KNN）
神经网络

## Unsupervised Learning (无监督学习)

K 均值聚类（K-Means）
层次聚类
高斯混合模型（GMM）
主成分分析（PCA）

## Reinforcement Learning (强化学习)

Q 学习（Q-Learning）
深度强化学习（Deep Q-Network，DQN）
